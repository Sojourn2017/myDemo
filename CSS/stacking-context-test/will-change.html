在 will-change 中指定了任意 CSS 属性，即便你没有直接指定这些属性的值

div触发层叠上下文，文字被背景色遮盖 
        <div style="background: #ccc;width: 200px;height: 200px;z-index: -2; will-change: transform;">
          <div style="position: absolute;z-index: -1;">触发触发层叠上下文，文字在背景色上方
        </div>
触发触发层叠上下文，文字在背景色上方

div未触发层叠上下文，文字被背景色遮盖 
        <div style="background: #ccc;width: 200px;height: 200px;z-index: -2;">
          <div style="position: absolute;z-index: -1;">触发触发层叠上下文，文字在背景色上方
        </div>
触发触发层叠上下文，文字在背景色上方
